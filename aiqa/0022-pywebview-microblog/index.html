    pywebviewでマイクロブログ

    [質問]

    pywebviewに興味があります。そのサンプルとして、個人で使うメモというかmicroblogを作ってください。画面の上部にフォームがあり、タイトルと本文を入れて書き込むと記事が追加されます。現在の記事は画面の下部にタイムラインとして表示されます。単語検索で見つかった記事のみ表示するフィルタ機能と、記事ごとに削除できるボタンも付けてください。

    [回答] Gemini

    pywebviewを使って、デスクトップアプリとして動作するシンプルなマイクロブログ（メモ帳）を作成しましょう。

    このアプリでは、データの保存に軽量な SQLite を使用し、フロントエンド（HTML/JS）とバックエンド（Python）を連携させます。

    必要な準備

    事前に pywebview をインストールしておいてください。

    pip install pywebview

    サンプルコード (app.py)

    import webview
import sqlite3
import os

# データベースの初期化
DB_NAME = "memo.db"

def init_db():
    with sqlite3.connect(DB_NAME) as conn:
        conn.execute("""
            CREATE TABLE IF NOT EXISTS posts (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                title TEXT,
                content TEXT,
                created_at DATETIME DEFAULT CURRENT_TIMESTAMP
            )
        """)

# Python側のロジック（API）
class Api:
    def get_posts(self, query=""):
        with sqlite3.connect(DB_NAME) as conn:
            conn.row_factory = sqlite3.Row
            cursor = conn.cursor()
            if query:
                cursor.execute("SELECT * FROM posts WHERE title LIKE ? OR content LIKE ? ORDER BY id DESC", 
                               (f'%{query}%', f'%{query}%'))
            else:
                cursor.execute("SELECT * FROM posts ORDER BY id DESC")
            return [dict(row) for row in cursor.fetchall()]

    def add_post(self, title, content):
        if not title or not content:
            return False
        with sqlite3.connect(DB_NAME) as conn:
            conn.execute("INSERT INTO posts (title, content) VALUES (?, ?)", (title, content))
        return True

    def delete_post(self, post_id):
        with sqlite3.connect(DB_NAME) as conn:
            conn.execute("DELETE FROM posts WHERE id = ?", (post_id,))
        return True

# UI (HTML/JavaScript/CSS)
html = """
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <style>
        body { font-family: sans-serif; margin: 20px; background: #f4f4f9; }
        .form-section { background: white; padding: 20px; border-radius: 8px; box-shadow: 0 2px 5px rgba(0,0,0,0.1); }
        input, textarea { width: 100%; margin-bottom: 10px; padding: 8px; box-sizing: border-box; }
        button { cursor: pointer; background: #007bff; color: white; border: none; padding: 10px 15px; border-radius: 4px; }
        button:hover { background: #0056b3; }
        .search-box { margin-top: 20px; padding: 10px; border: 1px solid #ddd; border-radius: 4px; width: 100%; }
        .timeline { margin-top: 20px; }
        .post { background: white; padding: 15px; margin-bottom: 10px; border-radius: 8px; border-left: 5px solid #007bff; position: relative; }
        .post h3 { margin-top: 0; }
        .delete-btn { position: absolute; top: 10px; right: 10px; background: #ff4d4d; font-size: 12px; }
    </style>
</head>
<body>
    <div class="form-section">
        <input type="text" id="title" placeholder="タイトル">
        <textarea id="content" rows="3" placeholder="今なにしてる？"></textarea>
        <button onclick="savePost()">投稿する</button>
    </div>

    <input type="text" id="search" class="search-box" placeholder="キーワードで検索..." oninput="loadPosts()">

    <div id="timeline" class="timeline"></div>

    <script>
        async function savePost() {
            const title = document.getElementById('title').value;
            const content = document.getElementById('content').value;
            const success = await pywebview.api.add_post(title, content);
            if (success) {
                document.getElementById('title').value = '';
                document.getElementById('content').value = '';
                loadPosts();
            }
        }

        async function loadPosts() {
            const query = document.getElementById('search').value;
            const posts = await pywebview.api.get_posts(query);
            const timeline = document.getElementById('timeline');
            timeline.innerHTML = '';
            posts.forEach(post => {
                const div = document.createElement('div');
                div.className = 'post';
                div.innerHTML = `
                    <h3>${post.title}</h3>
                    <p>${post.content}</p>
                    <small>${post.created_at}</small>
                    <button class="delete-btn" onclick="deletePost(${post.id})">削除</button>
                `;
                timeline.appendChild(div);
            });
        }

        async function deletePost(id) {
            if (confirm('削除しますか？')) {
                await pywebview.api.delete_post(id);
                loadPosts();
            }
        }

        // 初期読み込み
        window.addEventListener('pywebviewready', loadPosts);
    </script>
</body>
</html>
"""

if __name__ == "__main__":
    init_db()
    api = Api()
    window = webview.create_window("Micro Memo App", html=html, js_api=api, width=600, height=800)
    webview.start()

    このコードのポイント

    1. データの永続化: sqlite3 を使っているため、アプリを閉じてもメモは消えません。実行ディレクトリに memo.db が作成されます。
    2. 双方向の連携:
    3. Python側（Api クラス）でDB操作を定義。

    4. JavaScript側から pywebview.api.関数名() でそれを呼び出しています。

    5. リアルタイム検索: 検索窓（oninput）に入力するたびに loadPosts() が走り、Python側の LIKE 句を使ってフィルタリングされた結果を表示します。

    6. モダンなUI: CSSで少し整えて、デスクトップアプリらしい見た目にしています。
